# Medi-Backend

:white_check_mark: **Setup Checklist:** 
- NPM is installed 
- Node.js is installed  
- Git Bash is properly configured

### Instructions

1. Fork the repo
2. ```git clone <your-clone-link>```
3. ```cd ./Medi-Backend ```
4. ```npm install```
5. ```npm start```

Now open your browser and open ```localhost:8080```
